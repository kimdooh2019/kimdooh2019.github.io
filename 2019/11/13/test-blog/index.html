start blog

how to start blog

  1. node 설치

    • sudo apt update
    • sudo apt install nodejs

  2. npm 설치

    • sudo apt install npm

  3. hexo init [file name]

    • hexo init test-blog

  4. ls installed files

    =>

    • _config.yml
    • node_modules/
    • package.json
    • package-lock.json
    • scaffolds/
    • source/
    • themes/

    $ cd [file name]
    $ npm install
